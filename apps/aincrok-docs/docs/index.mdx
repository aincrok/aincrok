# Aincrok Documentation

Welcome to the Aincrok documentation! Aincrok is a next-generation AI coding agent built on the foundation of Roo/Cline/Kilo, enhanced with advanced features like MCP support, auto-planning, and improved performance.

## What is Aincrok?

Aincrok (Aincrok Is Not Cline Roo Or Kilo) is an AI-powered coding assistant VS Code extension that helps developers write, debug, and refactor code more efficiently. It combines the proven capabilities of popular AI coding tools with modern enhancements for a superior development experience.

## Key Features

- **<Icon name="bot" size={20} /> Advanced Auto-Planning**: Automatically breaks down complex tasks into manageable steps
- **<Icon name="plug" size={20} /> MCP Support**: Model Context Protocol integration for seamless tool connections  
- **<Icon name="zap" size={20} /> Enhanced Performance**: Optimized token management and context handling
- **<Icon name="wrench" size={20} /> Rich Tool Ecosystem**: Extensive collection of development tools
- **<Icon name="target" size={20} /> Smart Code Understanding**: Multi-file context awareness and intelligent suggestions
- **<Icon name="shield" size={20} /> Privacy First**: Your code stays private with flexible provider options

## What's New?

Aincrok brings several improvements over its predecessors:

- **Better Context Management**: Improved sliding window and token optimization
- **Enhanced Error Recovery**: More robust error handling and recovery mechanisms  
- **MCP Protocol**: Native support for Model Context Protocol tools
- **Performance Optimizations**: Faster response times and reduced resource usage
- **Extended Provider Support**: Works with more AI services out of the box
