# Setting up Aincrok

After installing Aincrok, you'll need to configure an AI provider to start coding with AI assistance. Aincrok supports multiple providers, giving you flexibility in choosing the best models for your needs and budget.

## Quick Setup Steps

1. **Open Aincrok:** Use `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) and search for "Aincrok"
2. **Choose your provider:** Select from OpenAI, Anthropic, Google, local models, and more
3. **Add your API key:** Enter your credentials for your chosen provider
4. **Start coding:** Begin your first AI-assisted development session

## Provider Selection

### <Icon name="zap" size={20} /> Popular Cloud Providers
- **OpenAI:** GPT-4 and GPT-3.5 models for general coding tasks
- **Anthropic:** Claude models for complex reasoning and analysis
- **Google:** Gemini models for code understanding and generation
- **Aincrok:** Our managed service with optimized models (if available)

### <Icon name="dollar-sign" size={20} /> Budget-Friendly Options
- **OpenRouter:** Access multiple models with competitive pricing
- **Local Models:** Use Ollama or LM Studio for free, private AI assistance
- **Groq:** High-speed inference at cost-effective rates

### <Icon name="shield" size={20} /> Privacy-Focused Options
- **Ollama:** Run models completely offline on your machine
- **LM Studio:** Local model management with privacy guarantees
- **Enterprise Providers:** AWS Bedrock, Google Vertex, Azure OpenAI

## Configuration Process

1. **Access Settings:** Click the settings gear icon in Aincrok's interface
2. **Provider Setup:** Navigate to "API Configuration" 
3. **Enter Credentials:** Add your API key and configure preferences
4. **Test Connection:** Verify your setup with a test request
5. **Save Configuration:** Store your settings for future use

## First Configuration

For your first setup, we recommend starting with one of these options:

### Option 1: OpenAI (Most Popular)
- Visit [OpenAI's website](https://openai.com) to create an account
- Generate an API key from your dashboard
- Add the key to Aincrok's OpenAI provider configuration

### Option 2: Ollama (Free & Private)
- Install [Ollama](https://ollama.com) on your machine
- Download a coding model like CodeLlama or Deepseek-Coder
- Configure Aincrok to use your local Ollama instance

### Option 3: Anthropic (Advanced Reasoning)
- Create an account at [Anthropic](https://anthropic.com)
- Generate API credentials
- Configure Claude models in Aincrok

You can configure multiple providers and switch between them based on your needs:
- **Development:** Use fast, cost-effective models for routine tasks
- **Complex Problems:** Switch to advanced models for challenging work
- **Privacy:** Use local models for sensitive code
